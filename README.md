# Business Intelligence Amazon Inventory Case Study

This repository contains a Business Intelligence case study report analysing Amazon’s data-driven inventory management approach.

The report discusses data requirements, data warehousing, ETL, OLAP, big data systems, cloud storage, and recommendations for inventory decision-making.
